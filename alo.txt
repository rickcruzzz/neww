ffNesta aula, demonstro como usar o GitHub para fazer alterações no projeto sem precisar do terminal. Acesse github.dev, edite arquivos, faça commits e push diretamente no navegador. Utilize o editor semelhante ao VS Code para gerenciar versões. Aprenda a deletar arquivos permanentemente e a descartar alterações. Visualize e gerencie commits feitos. A evolução da tecnologia traz soluções práticas. Estude e se especialize para acompanhar as novidades e melhorar suas habilidades de programação.
